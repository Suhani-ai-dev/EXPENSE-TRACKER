💰 Expense Tracker Pro
A modern and colorful Expense Tracker Web Application built using HTML, CSS, and JavaScript.
It helps users manage income and expenses with a clean UI, real-time calculations, and local storage support.

🌐 Live Demo
👉 https://suhani-ai-dev.github.io/EXPENSE-TRACKER/

✨ Features
➕ Add Income & Expenses

✏️ Edit Transactions

❌ Delete Transactions

🔍 Search Functionality

🏷️ Category Filtering

🌙 Dark Mode Support

📊 Live Balance, Income & Expense Summary

📅 Monthly Expense Tracking

📤 Export Data as CSV File

💾 Persistent Data using Local Storage

🛠 Tech Stack
HTML5

CSS3 (Gradient UI + Glassmorphism Design)

JavaScript (Vanilla JS DOM Manipulation)
